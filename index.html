HeyListen!

A discord bot for Spotify playback.

Add To Server Learn More

HeyListen! is a discord bot that assists with discord's integrated Spotify sharing. It allows the users of a discord channel limited control of the current Spotify sharer's playback, including adding tracks to the current queue and skipping the playing tack. HeyListen! comes with a simple permissions system that allows the sharer to tune how the channel can interact with playback.
